# 🧮 Calculator App (Python GUI)

## 📊 Project Overview

This is a modern Calculator application built using Python and Tkinter.
It supports basic arithmetic operations with both button clicks and keyboard input.

---

## 🚀 Features

* Basic operations (+, -, *, /)
* Clean modern UI (Dark Theme)
* Keyboard input support
* Clear (C) and Backspace (⌫) functionality
* Error handling

---

## 🛠️ Technologies Used

* Python 🐍
* Tkinter (GUI Library)

---

## 📂 Project Files

* `calculator.py` → Main Python file
* `Quick_calculator.exe` → Executable file (optional)

---

## ▶️ How to Run

### 🔹 Run Python File

```id="rzzb2f"
python calculator.py
```

### 🔹 OR Run EXE File

Just double click:

```id="kp9okg"
Quick_calculator.exe
```

---

## 🎮 Controls

* Numbers: 0–9
* Operators: +, -, *, /
* Enter → Calculate
* Backspace → Delete last digit
* Escape → Clear
