# 💰 Expense Tracker (Python CLI Project)

## 📊 Project Overview

This is a simple Expense Tracker application built using Python.
It allows users to record daily expenses, view all expenses, and calculate total spending.

---

## 🚀 Features

* Add new expenses
* View all expenses
* Calculate total expense
* Simple menu-driven interface (CLI)

---

## 🛠️ Technologies Used

* Python 🐍

---

## 📂 Project Structure

* `Expense_Tracker Project.py` → Main Python file

---

## ▶️ How to Run

1. Open terminal / command prompt
2. Run the script:

```
python "Expense_Tracker Project.py"
```

---

## 📌 How It Works

1. User selects option from menu
2. Adds expense details:

   * Date
   * Category
   * Description
   * Amount
3. Data stored in list
4. User can view or calculate total

---

## 📸 Sample Menu

```
====Menu====
1. Add Expenses
2. View All Expenses
3. View Total Cost
4. Exit
```
