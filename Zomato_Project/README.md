# 🍽️ Zomato Dataset Analysis Project

## 📌 Overview

This project analyzes restaurant data from Zomato using Python.
It uses data analysis and visualization techniques to understand customer preferences, restaurant trends, and ratings.

The dataset includes:

* Restaurant type
* Ratings
* Votes
* Cost for two people
* Online ordering availability

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas
* Matplotlib
* Seaborn

## ⚙️ Steps Performed

1. Data Import using Pandas
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Visualization using Matplotlib and Seaborn

---

## 📌 Key Insights

* Dining restaurants dominate in number
* Online ordering restaurants tend to have slightly better ratings
* Most ratings fall around 5 (average satisfaction)
* Cost for two people is mostly between ₹300–₹600
* Dining restaurants receive the highest votes
